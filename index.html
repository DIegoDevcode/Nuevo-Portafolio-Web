Platzi Travel

Este proyecto es una representación de un sitio web de reserva de viajes donde los usuarios pueden explorar destinos, y planificar sus próximas vacaciones. Este es el primer proyecto que tiene modo oscuro. Esta construido con el framework Tailwind css y html. Se uso las propiedades de css, display grid y flex, para la organización de las cards.
